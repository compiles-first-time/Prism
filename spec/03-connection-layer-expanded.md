## Purpose

Turn the Connection Layer architectural decisions from Spec 03 into implementation-ready SR rows. Covers: 8 connection types and their lifecycle, Normalized Execution Record, classification gate (synchronous Stages 1-2, asynchronous Stages 3-6), concurrency model (D-32 pull locks per scope), quarantine for failed classification (D-36), schema change detection (BP-04), connection KPIs (D-16), classification override map (BP-09), AI navigation governance.

## Architectural Decisions Covered

D-6 (tool-agnostic), D-8 (5 connection types + 2 indirect = 8 total via D-58), D-9 (AI nav vs API), D-16 (KPIs), D-32 (concurrency model), D-36 (quarantine), D-43 (cloud LLM as connection), D-54 (paywalled API policy), D-58 (Type 8 log stream), BP-01 (classification gate), BP-04 (schema change), BP-06 (Type 6), BP-07 (source schema version), BP-09 (override map), BP-25 (override persistence), BP-34 (rate limiting), BP-42 (Type 7), BP-62-66 (log ingestion).
